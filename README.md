# BriaCell Bria-ABC Event Model Site

This is a static GitHub Pages site.

## Structure

- `index.html` - website landing page
- `reports/en.html` - full English HTML report
- `reports/tr.html` - full Turkish HTML report
- `downloads/english-report.pdf` - English PDF
- `downloads/turkish-report.pdf` - Turkish PDF
- `assets/` - website charts and styles

## Deploy

Upload all files and folders in this directory to the repository root. Do not upload the parent folder itself.

GitHub Pages settings:

- Source: Deploy from a branch
- Branch: main
- Folder: / root
