# BriaCell Bria-ABC 144-Event Monte Carlo Report Site

This package is ready for GitHub Pages.

## Structure

- `index.html` - default English full report shown at the root URL
- `reports/en.html` - English full report
- `reports/tr.html` - Turkish full report
- `downloads/` - PDF downloads
- `.nojekyll` - prevents GitHub Pages/Jekyll processing issues

## Publish

1. Upload all files and folders in this directory to the root of your GitHub repository.
2. Go to Settings -> Pages.
3. Select Deploy from a branch.
4. Select branch `main` and folder `/root`.
5. Save.

Root URL will open the English report first. A language selector at the top links to the Turkish version.
